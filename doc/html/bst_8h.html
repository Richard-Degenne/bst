typedef int(* orderFun) (void *, void *)

Binary relation.

To work, the dataset of a BST must be ordered. This function type must return an int : a positive number if the first argument is greater than the second argument, and a negative number otherwise. A zero (0) return value will be considered as greater than, and will also be considered a valid result for a search.

void bst_destroy ( bst *  tree)

Destroys a tree.

A call to bst_destroy deallocates a tree and all its nodes by calling bst::free. Make sure the program calls it when the tree is not needed anymore.

Parameters
treeThe tree to deallocate
Warning
Not using this function to deallocate a bst will cause memory leaks!
